# RT快乐器

## 1. 这是什么？

这是一个快乐的项目，可以BT的兄弟们更好的去斗电子蛐蛐（RT）

## 2.怎么用？

当RT将dnslog的地址打到日志里的时候，拿着他的dnslog域名，给他一些极致的快乐。

## 3. 参数设置

~~~golang
-f  字典路径（空着的时候，会有内置字典）
-t  快乐次数，-1为究极快乐模式，无休止的快乐
-d  dnslog地址 
~~~
